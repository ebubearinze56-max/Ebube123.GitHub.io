# MovieHub

MovieHub is a web application that allows users to browse and download movies and pictures in high quality. The site features a clean and modern design, providing an easy-to-use interface for accessing a wide range of content.

## Project Structure

The project consists of the following files and directories:

- **index.html**: The main HTML structure for the MovieHub webpage, including the header, hero section, main content with movie listings, a download modal, and the footer.
- **assets/css/style.css**: Contains the CSS styles for the webpage, defining layout, colors, fonts, and responsive design.
- **assets/js/main.js**: Contains JavaScript code that adds interactivity to the webpage, including functionality for download buttons, filtering movies, and handling the search feature.
- **.gitignore**: Specifies files and directories to be ignored by Git, such as node_modules and logs.
- **CNAME**: Used to set a custom domain for the GitHub Pages site, containing the desired domain name.

## Getting Started

To get a local copy of the project up and running, follow these steps:

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/moviehub-github-pages.git
   ```

2. Navigate to the project directory:
   ```
   cd moviehub-github-pages
   ```

3. Open `index.html` in your web browser to view the project.
